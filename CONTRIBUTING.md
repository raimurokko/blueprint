# Contributing to this Project

First off, thank you for considering contributing to this project! 
It's people like you that make open source such a great community. 

## How to Contribute

1. **Fork** the repository and create your branch from `main`.
2. If you've added code that should be tested, please add tests.
3. Ensure your code passes all existing tests and linting.
4. **Sign off your commits** (see the DCO section below).
5. Issue a Pull Request!

---

## The Developer Certificate of Origin (DCO)

To ensure that all contributions to this project are legally safe 
and that you have the right to submit them, 
we use the **Developer Certificate of Origin (DCO)**. 

You do not need to sign any complex legal documents. 
All you need to do is add a *`Signed-off-by` *line to your Git commits. 

### How to sign your work

You can easily add this signature to your commit by using 
the `-s` (or `--signoff`) flag when you commit your changes:

```bash
git commit -s -m "Brief description of your changes"
