## 🤖 AI Transparency Notice

In the spirit of transparency, I want to disclose how Artificial Intelligence (AI) 
and Large Language Models (LLMs) are used in this project:

* **As a Core Feature:** This project actively utilizes AI models under the hood (e.g., via `litellm` and `chromadb`) to provide its main functionality, such as text generation and embeddings.
* **For Development & Coding:** AI tools were used as pair-programming assistants to write boilerplate code, generate standard algorithms, assist with debugging, make the code more readable, and generate comprehensive code comments.
* **Human Reviewed:** This project is a human-led initiative. All AI-generated code and suggestions were carefully reviewed, tested, and modified by a human developer before being merged.
